# Libraries Used
- **TFT_eSPI**
- **LGVL 9**
- Custom config files for **LGVL** and **TFT_eSPI**

# Pin List

## Screen Pins
| Pin       | GPIO    |
|-----------|---------|
| **SDO (MISO)**  | GPIO 12 |
| **LED**         | GPIO 21 |
| **SCK**         | GPIO 14 |
| **SDI (MOSI)**  | GPIO 13 |
| **D/C**         | GPIO 2  |
| **RESET**       | EN/RESET|
| **CS**          | GPIO 15 |
| **GND**         | GND     |
| **VCC**         | 3V      |
